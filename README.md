# How to start
## 1.clone repo to local
```
$ git clone  https://github.com/Hyman1993/home_task.git
```

## 2.clean and package and start
```
$ cd home_task
$ mvn clean
$ mvn package
$ cd target
$ java -jar home_task-0.0.1-SNAPSHOT.jar
```
